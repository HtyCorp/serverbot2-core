Link Sending Preferences
Authenticating

By default, commands like !addip send you links via Discord private message. Enabling browser notifications allows more convenient options to get these links.

This only sends notifications in response to commands, and can be disabled at any time.

Links will be sent as clickable browser notifications that open to the expected page.

!addip will automatically whitelist your IP address using your browser, with no extra manual steps.
